Python 3.12.0 release candidate 2 now available.

This release, 3.12.0rc2, is the last release preview for Python 3.12.There will be no ABI changes from this point forward in the 3.12 series. The intent is for the final release of 3.12.0, scheduled for Monday, 2023-10-02, to be identical to this release candidate. This is the last chance to find critical problems in Python 3.12.
